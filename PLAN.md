# PLAN.md — 旅行会社予約処理ワークフロー

## 背景・仕事の概要
旅行会社（エージェント）からメールでツアー予約依頼が入る。対象は決まったデイツアーのラインナップ（カスタムではない）。

## 登場するドキュメント・ツール

| ドキュメント/ツール | 用途 |
|---|---|
| 受信メール | 旅行会社からの予約依頼 |
| ツアーリストPDF | 扱うツアーの詳細（価格・内容・条件） |
| ガイド依頼書 | 各ツアーのガイドへの指示書 |
| コンファメーション | 旅行会社への正式確認書（ツアー1ヶ月前に発行） |
| ツアー管理Googleスプレッドシート | 予約一覧の管理 |
| Googleカレンダー | ツアー日程の管理 |
| ガイドアサイン表スプレッドシート | ガイドのアサイン管理 |
| ツアー原価表 | 各ツアーのコスト管理 |

## フロー全体像

### ステップ1：予約依頼受信後すぐ
1. 依頼内容の確認（日付・ツアー名・人数・特記事項）
2. 体験系ツアーの場合は予約可否の確認
3. 旅行会社へ返信（受付確認・請求予定金額・1ヶ月前にPDF送付する旨）
4. Googleカレンダーに登録
5. ツアー管理スプレッドシートに入力
6. ガイドアサイン表スプレッドシートに転記

### ステップ2：ツアー1ヶ月前
1. 予約詳細の再確認
2. ガイドのアサイン
3. コンファメーション作成・送付（ガイド名・集合場所を明記したPDF）
4. ガイド依頼書の作成・更新
5. 管理スプレッドシートの各ツアーフォルダにドキュメントを格納

## 料金ルール

| 区分 | 計算式 |
|---|---|
| 標準（AGT/BtoC） | タリフPDF記載の税抜単価 × 人数 × 1.1 |
| EXO | 税込単価 × 人数（PV加算なし） |
| プライベート（PV） | 上記 + 10,000円/ツアー |
| Attracxi系・EXO Dewa Sanzan | 人数スライド型（下記参照） |

### Attracxi系・EXO Dewa Sanzan スライド料金

| 人数 | 単価/人 |
|---|---|
| 1名 | 198,000円 |
| 2名 | 99,000円 |
| 3名 | 82,500円 |
| 4名 | 66,000円 |
| 5名～ | 60,500円 |
