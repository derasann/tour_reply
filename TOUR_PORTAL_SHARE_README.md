# ツアー予約割当ポータル（試用版）

社内で使用感を試してもらうための一時的な共有パッケージです。

## 使い方

### Mac
`run_tour_portal.command` をダブルクリックしてください。初回は自動でセットアップ（数分）が走った後、ブラウザでアプリが開きます。

### Windows
`run_tour_portal_windows.bat` をダブルクリックしてください。初回は自動でセットアップ（数分）が走った後、ブラウザでアプリが開きます。

Windowsで「発行元を確認できません」等の警告が出た場合は「詳細情報」→「実行」を選んでください。

## 注意事項

- **このパッケージにはAnthropic APIキーが同梱されています**（AI抽出機能のため）。課金は小野寺さんのアカウントに集約されます。試用が終わったら、このzip・展開したフォルダは削除してください。他の人に転送しないでください。
- このアプリは**予約データ（お客様名・アレルギー情報など）を保存しません**。生成した書類はその場でダウンロードするだけで、アプリを閉じると消えます。
- マスタデータ（ガイド・依頼元・ツアー・立ち寄り先単価など）は初回起動時にサンプルが自動で入りますが、各自の手元のフォルダ内だけに保存され、他の人とは共有されません。
- PDF書き出しには LibreOffice が必要です（無料）。入っていない場合は起動時に案内が表示されます。
  - Mac: `brew install --cask libreoffice` または https://www.libreoffice.org/download/download/
  - Windows: https://www.libreoffice.org/download/download/
- Excel・PowerPointの生成自体はLibreOfficeがなくても動作します（PDF書き出しのみ必要）。
